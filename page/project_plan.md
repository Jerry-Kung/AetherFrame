# 二次元美图开发引擎

## 1. 项目描述
轻量级个人图像处理工具，面向喜爱二次元风格的用户，提供素材加工、美图创作、图片修补三大功能模块。整体风格可可爱爱、柔和精致，主打简洁易用的单页面工具体验。

## 2. 页面结构
- `/` - 首页（模式切换入口）

## 3. 核心功能
- [x] 首页 Header + 模式切换 + 内容展示框架
- [ ] 素材加工模块（图像裁剪、格式转换、批量处理等）
- [ ] 美图创作模块（AI 绘图、风格化生成等）
- [ ] 图片修补模块（去水印、背景抹除、修复等）

## 4. 数据模型设计
暂无数据库需求，当前为纯前端工具。

## 5. 后端 / 第三方集成计划
- Supabase：暂不需要
- Shopify：不需要
- Stripe：不需要
- 后续可考虑：图像处理 API（如 Replicate、Cloudinary 等）

## 6. 开发阶段计划

### Phase 1：首页框架搭建
- 目标：完成三模式切换的首页 UI，建立整体视觉风格
- 交付物：可交互的模式切换界面，带空状态占位

### Phase 2：素材加工模块
- 目标：接入素材加工业务功能
- 交付物：完整的素材加工操作页面

### Phase 3：美图创作模块
- 目标：接入图像生成 / 风格化功能
- 交付物：美图创作操作页面

### Phase 4：图片修补模块
- 目标：接入图像修复功能
- 交付物：图片修补操作页面
